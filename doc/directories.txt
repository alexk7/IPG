+ ipg               // Incremental Parser Generator: Root directory for the whole project
\-+ src             // Source: Everything needed to build the project, read-only when building binaries
  |-+ cpp           // C++: All C++ source code files
  | |-+ Containers
  | |-+ PT          // Parse Tree
  | |-+ AST         // Abstract Syntax Tree
  | \-+ PTToAST     // PT to AST: Module for conversion from a PT to an AST

    |-+ vs2008      // Visual Studio 2008
    \-+ vs2010      // Visual Studio 2010
